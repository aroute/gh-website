Day 2 operation: Storage and container image registry setup​

For storage, we will opt for Rook/Ceph technology and use it in a manner suitable for the demo use case.

  1. After the cluster is installed and you are logged in, create Rook/Ceph cluster.
oc apply -f https://raw.githubusercontent.com/rook/rook/master/deploy/examples/crds.yaml
oc apply -f https://raw.githubusercontent.com/rook/rook/master/deploy/examples/common.yaml
oc apply -f https://raw.githubusercontent.com/rook/rook/master/deploy/examples/operator-openshift.yaml
oc apply -f https://raw.githubusercontent.com/rook/rook/master/deploy/examples/cluster-test.yaml
while [ "$CEPHCLUSTER" != "Ready" ]; do CEPHCLUSTER=$(oc get CephCluster -n rook-ceph my-cluster -o jsonpath="{.status.phase}{'\n'}"); echo "Setting up Rook Ceph Storage..." $CEPHCLUSTER; sleep 20; done
oc apply -f https://raw.githubusercontent.com/rook/rook/release-1.12/deploy/examples/filesystem-test.yaml
while [ "$CEPHFILESYSTEM" != "Ready" ]; do CEPHFILESYSTEM=$(oc get CephFilesystem -n rook-ceph myfs -o jsonpath="{.status.phase}{'\n'}"); echo "Setting up Rook File System..." $CEPHFILESYSTEM; sleep 20; done
oc apply -f https://raw.githubusercontent.com/rook/rook/master/deploy/examples/csi/rbd/storageclass-test.yaml
oc apply -f https://raw.githubusercontent.com/rook/rook/master/deploy/examples/csi/cephfs/storageclass.yaml
oc apply -f https://raw.githubusercontent.com/rook/rook/release-1.12/deploy/examples/object-test.yaml
oc apply -f https://raw.githubusercontent.com/rook/rook/release-1.12/deploy/examples/storageclass-bucket-delete.yaml
oc expose svc/rook-ceph-mgr-dashboard -n rook-ceph
oc expose svc/rook-ceph-rgw-my-store -n rook-ceph
  1. Create OpenShift Container Image Registry Server.
cat <<EOF | oc create -f -
kind: PersistentVolumeClaim
apiVersion: v1
metadata:
  annotations:
  name: image-registry-storage
  namespace: openshift-image-registry
spec:
  accessModes:
    - ReadWriteMany
  resources:
    requests:
      storage: 100Gi
  storageClassName: rook-cephfs
  volumeMode: Filesystem
EOF
oc patch config.image/cluster -p '{"spec":
{"managementState":"Managed","replicas":1,"storage":
{"managementState":"Unmanaged","pvc":{"claim":"registry-storage-pvc"}}}}' --type=merge
